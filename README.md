4. Как с помощью NumPy найти ковариацию двух переменных?

    С помощью функции `numpy.cov`. Например:
    ```python
    import numpy as np
    a = np.array([[1, 2, 1, 3], [5, 3, 1, 8]], float)
    np.cov(a)
    
    # output:
    [[0.91666667 2.08333333]
      [2.08333333 8.91666667]]
    ```
